欢迎使用 Auth Service 管理后台

这是一个基于 Spring Boot 3.4 + JWT + RBAC 的认证服务管理系统。

  • 支持多租户隔离
  • 基于角色的访问控制 (RBAC)
  • JWT 双令牌认证机制
  • 平台级权限管理
